<h1>EventGather</h1>

<h3>Описание:</h3>
Приложение предназначено для создания афиш событий и сбора участников. 
Пользователь может создать свое событие или принять участие в нем.
<h3>Стек:</h3>
<ul>
  <li>JavaCore</li>
  <li>Git</li>
  <li>Docker</li>
  <li>Hibernate</li>
  <li>Spring Boot</li>
  <li>Microservice architecture</li>
  <li>REST API</li>
</ul>
